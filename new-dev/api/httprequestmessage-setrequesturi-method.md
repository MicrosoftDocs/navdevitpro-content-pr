# SetRequestUri Method
Sets the Uri used for the HTTP request.

```
[Ok := ] HttpRequestMessage.SetRequestUri(RequestUri)
```

## Parameters
*HttpRequestMessage*  
&emsp;Type: HttpRequestMessage

*RequestUri*  
&emsp;Type: Text

## Return Value
*Ok*  
&emsp;Type: Boolean  
&emsp;**true** if the Uri was valid; otherwise, **false**.
