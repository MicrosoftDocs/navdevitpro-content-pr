# IsUndefined Method

Indicates whether the JsonValue contains the JSON value of UNDEFINED.

```
Ok := JsonValue.IsUndefined
```

### Parameters
*JsonValue*  
&emsp;Type: JsonValue

## Property Value/Return Value
Type: Boolean

**true** if the JsonValue contains the JSON value of UNDEFINED; otherwise, **false**.
