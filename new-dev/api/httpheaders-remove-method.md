# Remove Method
Removes the key and the related values from the HttpHeaders object.

```
[Ok := ] HttpHeaders.Remove(Key)
```

## Parameters
*HttpHeaders*  
&emsp;Type: HttpHeaders

*Key*  
&emsp;Type: Text

## Return Value
&emsp;Type: Boolean  
&emsp;**true** if key exists; **false** otherwise.
