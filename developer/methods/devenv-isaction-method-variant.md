# ISACTION Method (Variant)
Indicates whether an AL variant contains an Action variable.  
  
## Syntax  
  
```  
  
Ok := Variant.ISACTION  
```  
  
#### Parameters  
 *Variant*  
 Type: Variant  
  
## Property Value/Return Value  
 Type: Boolean  
  
 **true** if the AL variant contains an Action variable; otherwise, **false**.
