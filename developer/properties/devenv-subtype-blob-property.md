# SubType Property (BLOB)
Sets additional information about what will be contained in the field.  
  
## Applies To  
 BLOB fields  
  
## Property Value  
  
|**Value**|**Description**|  
|---------------|---------------------|  
|**Bitmap**|For storing bitmaps|  
|**Memo**|For storing memos|  
|**User-Defined \(default\)**|For user defined information|
