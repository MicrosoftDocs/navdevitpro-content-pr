# DataLength Data Type
Sets the maximum length of data stored in this field.  
  
## Applies To  
 Fields  
  
## Remarks  
 For [Text Data Type](devenv-text-data-type.md) and [Code Data Type](devenv-code-data-type.md) fields, enter the maximum length of the field. The database only stores the actual content of the fieldâ€”not the maximum length you specify.  
  
 This field does not apply to the other data types because they have fixed lengths.
