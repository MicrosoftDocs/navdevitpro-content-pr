# Add Method
Adds sets the provided value for the provided header name.

```
[Ok := ] HttpHeaders.Add(Key, Value)
```

## Parameters
*HttpHeaders*  
&emsp;Type: HttpHeaders

*Key*  
&emsp;Type: Text

*Value*  
&emsp;Type: Text

## Return Value
*Ok*  
&emsp;Type: Boolean  
&emsp;**True** if the operation was successful; otherwise, **false**.
