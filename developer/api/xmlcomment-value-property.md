# Value Property
Gets or sets the string value of this comment.  
```  
[Value := ] XmlComment.Value([NewValue])  
```  
## Parameters
*NewValue*    
&emsp;Type: String  
The new value of this comment.  
  
## Property Value
*Value*  
&emsp;Type: String
