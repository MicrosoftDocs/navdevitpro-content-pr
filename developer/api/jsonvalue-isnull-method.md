# IsNull Method

Indicates whether the JsonValue contains the JSON value of NULL.

```
Ok := JsonValue.IsNull
```

## Parameters
*JsonValue*  
&emsp;Type: JsonValue

## Property Value/Return Value
Type: Boolean

**True** if the JsonValue contains the JSON value of NULL; otherwise, **false**.
