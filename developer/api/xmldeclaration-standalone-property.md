# Standalone Property
Gets or sets the standalone property for this document.  
```  
[Value := ] XmlDeclaration.Standalone([NewValue])  
```  
## Parameters
*NewValue*    
&emsp;Type: String  
A string containing the standalone property for this document.  
  
## Property Value
*Value*  
&emsp;Type: String
