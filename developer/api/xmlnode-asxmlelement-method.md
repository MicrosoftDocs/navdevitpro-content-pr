# AsXmlElement Method
Converts the node to an XmlElement node. The operation will fail if the node is not an XmlElement.  
```  
XmlElement := XmlNode.AsXmlElement()  
```  
## Return Value
*XmlElement*  
&emsp;Type: XmlElement
