# Add Method
Adds a new value at the end of the JsonArray.

```
JsonArray.Add(Value)
```

## Parameters
*JsonArray*  
&emsp;Type: JsonArray

*Value*  
&emsp;Type: JsonToken | JsonObject | JsonValue | JsonArray | Boolean | Char | Byte | Integer | BigInteger | Decimal | Duration | String | Date | Time | DateTime
