# AsXmlDocumentType Method
Converts the node to an XmlDocumentType node. The operation will fail if the node is not an XmlDocumentType.  
```  
XmlDocumentType := XmlNode.AsXmlDocumentType()  
```  
## Return Value
*XmlDocumentType*  
&emsp;Type: XmlDocumentType
