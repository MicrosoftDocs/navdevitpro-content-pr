# IsObject Method

Indicates whether a JsonToken represents/contains a JSON object.

```
Ok := JsonToken.IsObject
```

### Parameters
*JsonToken*
Type: JsonToken

## Property Value/Return Value
Type : Boolean

**true** if the JsonToken represents a JSON object; otherwise, **false**.
