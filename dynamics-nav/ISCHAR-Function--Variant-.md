# ISCHAR Function (Variant)
Indicates whether a C/AL variant contains a Char variable.  
  
## Syntax  
  
```  
  
Ok := Variant.ISCHAR  
```  
  
#### Parameters  
 *Variant*  
 Type: Variant  
  
## Property Value/Return Value  
 Type: Boolean  
  
 **true** if the C/AL variant contains a Char variable; otherwise, **false**.
