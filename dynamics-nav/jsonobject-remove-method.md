# Remove Method
Removes the property with the given key from the object.

```
[Ok := ] JsonObject.Remove(Key)
```

## Parameters
*Key*  
&emsp;Type: String

## Return Value  
Type: Boolean

**true** if a property with the given key was present on the object; otherwise, **false**.
