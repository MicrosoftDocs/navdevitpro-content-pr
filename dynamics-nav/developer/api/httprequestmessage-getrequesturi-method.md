# GetRequestUri Method
Gets Uniform Resource Identifier (URI) used for the HTTP request.

```
RequestUri := HttpRequestMessage.GetRequestUri
```
## Parameters
*HttpRequestMessage*  
&emsp;Type: HttpRequestMessage

## Return Value
*RequestUri*  
&emsp;Type: String
