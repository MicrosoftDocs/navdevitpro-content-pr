# AsXmlCData Method
Converts the node to an XmlCData node. The operation will fail if the node is not an XmlCData.  
```  
XmlCData := XmlNode.AsXmlCData()  
```  
## Return Value
*XmlCData*  
&emsp;Type: [XmlCData](xmlcdata-class.md)
