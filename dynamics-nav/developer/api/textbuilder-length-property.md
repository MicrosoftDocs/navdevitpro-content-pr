# Length Property
Gets or sets the length of the current TextBuilder object.  
```  
[OldLength := ] TextBuilder.Length([NewLength])  
```  
## Parameters
*NewLength*    
&emsp;Type: Integer  
The new length of this instance.  
  
## Property Value
*OldLength*  
&emsp;Type: Integer
