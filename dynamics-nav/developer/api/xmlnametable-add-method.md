# Add Method
Atomizes the specified string and adds it to the NameTable.  
```  
[AddedKey := ] XmlNameTable.Add(Key)  
```  
## Parameters
*Key*    
&emsp;Type: String  
The string to add.  
  
## Return Value
*AddedKey*  
&emsp;Type: String
