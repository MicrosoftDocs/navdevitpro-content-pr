# Create Method
Creates an XmlDocument.  
```  
XmlDocument := XmlDocument.Create(Content,...)  
```  
## Parameters
*Content*    
&emsp;Type: Joker  
The content to add to this document.  
  
## Return Value
*XmlDocument*  
&emsp;Type: XmlDocument
