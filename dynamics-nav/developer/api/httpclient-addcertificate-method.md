# AddCertificate Method

```
HttpClient.AddCertificate(certificate, password)
```
## Parameters
*HttpClient*  
&emsp;Type: [HttpClient](httpclient-class.md)

*certificate*  
&emsp;Type: [Text](/datatypes/devenv-text-data-type.md)

*password*  
&emsp;Type: [Text](/datatypes/devenv-text-data-type.md)

## Remarks

The certificate must be in base 64 format.
