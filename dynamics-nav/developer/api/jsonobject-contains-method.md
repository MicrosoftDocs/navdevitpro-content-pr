# Contains Method
Verifies if a JsonObject contains a property with a given key.

```
Ok := JsonObject.Contains(String)
```

## Parameters
*JsonObject*  
&emsp;Type: JsonObject

*String*  
&emsp;Type: String

## Return Value
Type: Boolean

**True** if the object contains a property with the given key; otherwise, **false**.
