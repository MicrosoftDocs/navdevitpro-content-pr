# Remove Method
Removes the specified attribute from the collection.  
```  
XmlAttributeCollection.Remove(Name)  
```  
## Parameters
*Name*    
&emsp;Type: [String](/datatypes/devenv-text-data-type.md)  
The qualified name of the attribute to remove.
