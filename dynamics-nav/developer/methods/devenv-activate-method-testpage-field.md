# ACTIVATE Method (TestPage Field)
Activates a field on a test page.  
  
## Syntax  
  
```  
TestPage.ACTIVATE  
```  
  
#### Parameters  
 *TestPage*  
 Type: TestPage  
  
 The test page variable that you use to refer to the test page.
