# ISBINARY Method (Variant)
Indicates whether an AL variant contains a Binary variable.  
  
## Syntax  
  
```  
  
Ok := Variant.ISBINARY  
```  
  
#### Parameters  
 *Variant*  
 Type: Variant  
  
## Property Value/Return Value  
 Type: Boolean  
  
 **True** if the AL variant contains a Binary variable; otherwise, **false**.
