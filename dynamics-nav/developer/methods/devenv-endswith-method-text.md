# ENDSWITH Method
Determines whether the end of this string instance matches the specified string.  
```  
Ok := Text.ENDSWITH(Value)  
```  
## Parameters
*Value*    
&emsp;Type: Text  
The string to compare to the substring at the end of this instance.  
  
## Return Value
*Ok*  
&emsp;Type: Boolean
