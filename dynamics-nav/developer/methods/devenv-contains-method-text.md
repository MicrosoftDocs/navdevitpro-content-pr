# CONTAINS Method
Returns a value indicating whether a specified substring occurs within this string.  
```  
Ok := Text.Contains(Value)  
```  
## Parameters
*Value*    
&emsp;Type: Text  
The string to seek.  
  
## Return Value
*Ok*  
&emsp;Type: Boolean
