# EXPAND Method (TestPage)
Expands rows on a test page.  
  
## Syntax  
  
```  
MyTestPage.EXPAND(Expand);  
```  
  
#### Parameters  
 *TestPage*  
 Type: TestPage  
  
 The test page that contains the rows that you want to expand.  
  
 *Expand*  
 Type: row  
  
 The rows that you want to expand on the test page.
