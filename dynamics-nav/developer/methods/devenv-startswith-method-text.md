# STARTSWITH Method
Determines whether the beginning of this instance matches a specified string.  
```  
Ok := Text.STARTSWITH(Value)  
```  
## Parameters
*Value*    
&emsp;Type: Text  
The string to compare.  
  
## Return Value
*Ok*  
&emsp;Type: Boolean
