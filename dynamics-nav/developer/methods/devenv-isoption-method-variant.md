# ISOPTION Method (Variant)
Indicates whether an AL variant contains an Option variable.  
  
## Syntax  
  
```  
  
Ok := Variant.ISOPTION  
```  
  
#### Parameters  
 *Variant*  
 Type: Variant  
  
## Property Value/Return Value  
 Type: Boolean  
  
 **True** if the AL variant contains an Option variable; otherwise, **false**.
