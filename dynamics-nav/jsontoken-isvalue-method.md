# IsValue Method

Indicates whether a JsonToken represents a JSON value.

```
Ok := JsonToken.IsValue
```

### Parameters
*JsonToken*  
&emsp;Type: JsonToken

## Property Value/Return Value
Type : Boolean

**true** if the JsonToken represents a JSON value; otherwise, **false**.
