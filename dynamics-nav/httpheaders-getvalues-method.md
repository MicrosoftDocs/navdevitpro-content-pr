# GetValues Method
Gets the values for the specified key.

```
[Ok := ] HttpHeaders.GetValues(Key, Result)
```

## Parameters
*HttpHeaders*  
&emsp;Type: HttpHeaders

*Key*  
&emsp;Type: Text

*Result*  
&emsp;Type: Array of Text

## Return Value
*Ok*  
&emsp;Type: Boolean  
&emsp;**true** if headers contain the key; **false** otherwise.
