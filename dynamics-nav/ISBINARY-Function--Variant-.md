# ISBINARY Function (Variant)
Indicates whether a C/AL variant contains a Binary variable.  
  
## Syntax  
  
```  
  
Ok := Variant.ISBINARY  
```  
  
#### Parameters  
 *Variant*  
 Type: Variant  
  
## Property Value/Return Value  
 Type: Boolean  
  
 **true** if the C/AL variant contains a Binary variable; otherwise, **false**.
