# Count Method
Gets the number of elements in the JsonArray.

```
Integer := JsonArray.Count
```

### Parameters
*JsonArray*  
&emsp;Type: JsonArray

## Property Value/Return Value
Type: Integer
The number of elements in the JsonArray.
