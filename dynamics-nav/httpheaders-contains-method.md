# Contains Method
Checks if an HttpHeaders contains a property with a given key.

```
Result := HttpHeaders.Contains(Key)
```

## Parameters
*HttpHeaders*  
&emsp;Type: HttpHeaders

*Key*  
&emsp;Type: Text

## Return Value
*Result*  
&emsp;Type: Boolean  
&emsp;**true** if the object contains a property with the given key; otherwise, **false**.
