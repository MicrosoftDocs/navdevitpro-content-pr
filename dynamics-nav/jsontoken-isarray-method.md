# IsArray Property
Indicates whether a JsonToken represents a JSON array.

```
Ok := JsonToken.IsArray
```

### Parameters
*JsonToken*  
&emsp;Type: JsonToken

## Property Value/Return Value
Type: Boolean

**true** if the JsonToken represents a JSON array; otherwise, **false**.
