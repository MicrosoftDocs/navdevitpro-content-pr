# DataLength Property
Sets the maximum length of data stored in this field.  
  
## Applies To  
 Fields  
  
## Remarks  
 For [Text Data Type](Text-Data-Type.md) and [Code Data Type](Code-Data-Type.md) fields, enter the maximum length of the field. The database only stores the actual content of the fieldâ€”not the maximum length you specify.  
  
 This field does not apply to the other data types because they have fixed lengths.
