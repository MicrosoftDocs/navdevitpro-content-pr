# ISOPTION Function (Variant)
Indicates whether a C/AL variant contains an Option variable.  
  
## Syntax  
  
```  
  
Ok := Variant.ISOPTION  
```  
  
#### Parameters  
 *Variant*  
 Type: Variant  
  
## Property Value/Return Value  
 Type: Boolean  
  
 **true** if the C/AL variant contains an Option variable; otherwise, **false**.
