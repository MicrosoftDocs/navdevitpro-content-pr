# ISACTION Function (Variant)
Indicates whether a C/AL variant contains an Action variable.  
  
## Syntax  
  
```  
  
Ok := Variant.ISACTION  
```  
  
#### Parameters  
 *Variant*  
 Type: Variant  
  
## Property Value/Return Value  
 Type: Boolean  
  
 **true** if the C/AL variant contains an Action variable; otherwise, **false**.
