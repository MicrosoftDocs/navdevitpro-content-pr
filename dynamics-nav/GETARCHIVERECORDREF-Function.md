# GETARCHIVERECORDREF Function
Returns a RecordRef for the specified table.  
  
## Syntax  
  
```  
[Ok :=] GETARCHIVERECORDREF(TableNo, RecordRef)  
```  
  
#### Parameters  
 *TableNo*  
 Type: Integer  
  
 Specifies the table ID.  
  
 *RecordRef*  
 Type: Variable  
  
 Specifies the reference.
